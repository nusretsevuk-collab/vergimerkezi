# vergimerkezi

OTMAHER Vergi Hesaplama Modülü için başlangıç arayüzü.

## Özellikler
- Manuel ciro girişi (KDV dahil → KDV hariç ve satış KDV ayrıştırma)
- Gider faturası girişi (%10/%20 KDV ayrıştırma)
- Manuel gider girişi
- Dashboard metrikleri:
  - Toplam ciro
  - KDV hariç ciro
  - Toplam gider
  - İndirilecek KDV
  - Ödenecek KDV
  - Net ticari kâr
  - Tahmini geçici vergi
- Günlük satış, gider ve 3 aylık vergi paneli
- localStorage ile veri saklama

## Çalıştırma
Tarayıcıda `index.html` dosyasını açmanız yeterlidir.
